# AI-Based-Resume-Shortlisting-System-NLP-ML-
AI-powered resume screening and candidate ranking system using NLP and Machine Learning. Built with Python and Flask to automate resume parsing, skill extraction, job description matching, and candidate ranking using TF-IDF and cosine similarity for efficient recruitment.

## Overview

The AI-Based Resume Shortlisting System is an intelligent recruitment solution that automates resume screening and candidate ranking using Natural Language Processing (NLP) and Machine Learning techniques. The system analyzes resumes, extracts skills, compares them with job descriptions, and ranks candidates based on relevance scores.

---

## Features

* Resume parsing and text extraction
* Skill extraction using NLP
* Candidate ranking using TF-IDF and cosine similarity
* Job description matching
* Automated resume screening process
* Recruiter-friendly web interface
* Fast and efficient candidate evaluation

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Resume-Shortlisting-System.git
```

### 2. Navigate to Project Folder

```bash
cd AI-Resume-Shortlisting-System
```

### 3. Create Virtual Environment

```bash
python -m venv venv
```

### 4. Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Run the Application

```bash
python app.py
```

## How It Works

1. Upload candidate resumes
2. Enter job description
3. System extracts skills and keywords
4. TF-IDF and cosine similarity calculate matching score
5. Candidates are ranked automatically
